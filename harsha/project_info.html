USB Physical Security Application

This application provides a basic layer of physical security for your computer's USB ports, primarily focusing on USB mass storage devices.

Key Features:

  • USB Port Control: Enable or disable USB mass storage functionality on your Windows system.
  • Password Protection: Critical actions (enable/disable) require a one-time password (OTP) sent to your registered email address.
  • Activity Logging: All enable/disable actions are logged with timestamps and success status.
  • Real-time Status: View the current status of USB mass storage (enabled/disabled).
  • Intruder Detection: In case of an incorrect password attempt, the application can record a short video using your webcam.

How it Works:

The application interacts with the Windows Registry to modify the 'Start' value of the USBSTOR service. This service controls the Plug and Play functionality for USB mass storage devices.

  • Setting 'Start' to 4 effectively disables the service, preventing new USB drives from being recognized.
  • Setting 'Start' to 3 re-enables the service.
Important Note:

This application primarily affects newly connected USB mass storage devices (like pendrives, external hard drives). It does not typically disable existing connected devices, USB keyboards, mice, or webcams. For a more comprehensive disable of all USB functionality, advanced system tools (like Microsoft's DevCon utility) would be required, which is beyond the scope of this self-contained Python application.

Administrator privileges are required to run this application and modify USB settings.

For email functionality, if you are using Gmail, you will need to generate an "App password" instead of using your regular Gmail password. Refer to Google's documentation on "App passwords" for more details.
